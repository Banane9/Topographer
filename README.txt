<< About >>

Topographer is a program for generating overhead maps of Minecraft worlds.
Currently only the Anvil format, introduced in Minecraft 1.2, is supported.

TopographerUI.exe provides a graphical user interface while TopographerCMD.exe
provides a command line interface.
